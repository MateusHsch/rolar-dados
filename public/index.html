Rolar Dados

Aqui você pode rolar diferentes tipos de dados usando prompts na barra abaixo.
Alguns exemplos de rolagem que você pode fazer:

Rolar 3d10 e somar os resultados => 3d10
Rolar 2d20 com vantagem e somar 5 => +2d20+5
Rolar 3d20 com desvantagem e subtrair 3 => -3d20-3
Estrutura => [van/des][n° dados][d][tipo dado][mod]
